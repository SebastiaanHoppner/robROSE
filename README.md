# robROSE
This repository contains the paper and the corresponding R package for robROSE: An approach for dealing with imbalanced data in fraud detection by B. Baesens, S. HÃ¶ppner, I. Ortner and T. Verdonck (2020). The R package `robROSE` contains the implementation of the robROSE algorithm for robust oversampling of minority cases.

The R package can be installed through the `devtools` package:  
`devtools::install_github("SebastiaanHoppner/robROSE/robROSE")`
